# @deveox/storage

This library ease interaction with Web Storage

- Lightweight
- Fully tree-shakable: import only what you need
- TypeScript: entirely typed
- `BigInt` support
- Test coverage > 95%
- Hight standards of code quality and contribution

## Installation

```shell
npm i @deveox/storage

```
